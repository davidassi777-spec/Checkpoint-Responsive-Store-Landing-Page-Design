Le site utilise Bootstrap 5 pour le responsive  la grille (col-12 col-sm-6 col-lg-3) adapte automatiquement les 4 cartes produits selon la taille d'écran (1 colonne sur mobile, 2 sur tablette, 4 sur desktop). La navbar Bootstrap gère le menu hamburger sur mobile. Le CSS personnalisé se limite à la bannière d'accueil (image de fond, texte lisible) et une media query pour ajuster les tailles sur petits écrans.
